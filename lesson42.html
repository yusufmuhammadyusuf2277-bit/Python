Lesson 42: Safe Port Scanner

This script scans ports on your own computer for learning purposes.

import socket

target = "127.0.0.1"
ports = [21, 22, 80, 443]

for port in ports:
  s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
  result = s.connect_ex((target, port))
  if result == 0:
    print(f"Port {port} is open")
  s.close()

Understanding open ports helps you secure your own system.
