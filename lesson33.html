Lesson 33: Password List Checker

This script checks passwords against a weak-password list.

weak_passwords = ["123456", "password", "admin"]
user_password = input("Enter password: ")

if user_password in weak_passwords:
  print("Weak password detected")
else:
  print("Password looks safe")

Used for password auditing and awareness.
