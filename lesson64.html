Lesson 64: Local Network Mapper

Scan your local network to see which devices are online (safe, your own network).

import os
import ipaddress

network = ipaddress.ip_network("192.168.1.0/30")
for ip in network.hosts():
  res = os.system(f"ping -c 1 {ip}")
  status = "Online" if res==0 else "Offline"
  print(f"{ip}: {status}")

Learning network mapping is essential for security monitoring.
